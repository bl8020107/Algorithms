

-----------------תיאור -------------

פרויקט קטן המכיל מספר קבצי Java  אלגוריתם למציאת תת־מחרוזות מגדילות, תכנון מודלים (DTOs) למערכת ספרייה ותיעוד רעיוני לניהול הזמנות ותשלומים.

קבצים עיקריים

- `Array.java` – מחלקה `IncreasingSubarrays` עם פונקציה שמוצאת רשימות משנה של איברים עוקבים שמגדילים.
- `Library.java` – תיעוד וקונספט של API למערכת ספרייה (Endpoints, DTOs, שדות חובה וכו').
- `Orders.java` – תיעוד רעיוני לניהול הזמנות, פריטים ותשלומים.





---------------דרכי הרצה (Windows - PowerShell)

לקמפל ולהריץ קובץ יחיד:

- לקמפל: `javac Array.java`
- להריץ: `java IncreasingSubarrays`

להריץ פרויקט שלם (כל הקבצים):

- לקמפל את כל הקבצים: `javac *.java`
- להריץ מחלקה שמכילה `main` (אם תיצור אחת): `java ClassName`

הערות חשובות

- הקבצים `Library.java` ו־`Orders.java` הם תיעוד/סקיצה ולא מכיל קבצים פעילים.
